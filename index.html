What the heck is the event loop anyway?

Hybrid Environment

  • Sync & async ops.
  • Sync: Straight to stack, executed immediately.
  • Async: Executed off-stack, multi-threaded. callback(result) to callback queue.
  • When stack is empty, event loop moves callbacks from queue to stack.
  • This is why node.js is fast.

So how to call async functions sequentially?

An evolution of approaches...

Nested Callbacks

function1(function (err, result1) {
  (err ? console.log(err) : 
    function2(result1, function(err, result2) {
      (err ? console.log(err) :
        function3(result2, function(err, result3) {
          (err ? console.log(err) :
            function4(result3);  
          ); 
        });          
      );
    });
  );
});
